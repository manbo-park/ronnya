# 론냐🐾 ronnya

리치마작 룰 학습·실력 향상을 위한 모바일 웹 앱.

홈에서 주요 메뉴(연습하기·계산하기)를 전면에 두고, 참고용 메뉴는 **더 알아보기** 그룹으로 묶어 제공합니다.

- [연습하기](#연습하기) (`/train`) — 화료 점수 맞추기
- [계산하기](#계산하기) (`/calc`) — 부수/판수 직접 입력 또는 패 입력으로 점수 계산
- **더 알아보기**
    - [점수표](#점수표) (`/score`) — 판·부수별 점수 조견표
    - [역 목록](#역-목록) (`/yaku`) — 역 이름·설명·예시
    - 부수 계산법 (`/fu`) — 부수 항목별 계산 안내

## 연습하기

화료 시점의 손패·후로·오름패와 상황(장풍/자풍, 쯔모·론, 리치 등)을 보고 점수를 맞추는 게임입니다.

- 매번 랜덤 문제 생성 (구성적 생성 + 채점 엔진 검증으로 불가능한 케이스 차단)
- 도전 Phase: 점수 입력 → 확인
- 해답 Phase: x판 y부 z점, 역별 판수, 부수 계산 상세, 정오 표시
- 0본장 고정

## 계산하기

점수·부수를 계산하는 도구로, 두 방식의 탭을 제공합니다.

- **부수/판수로 계산하기** 탭: 화료 형태·대기·또이쯔·커쯔 등 부수 항목을 직접 골라 점수와 부수를 계산
    - '부수만 계산하기' 체크 시 자/친·판수 입력 없이 부수만 계산
    - 핑후·치또이 등 특수 형태 지원, 모순된 입력은 자동 잠금·보정
    - 역 판정·도라 계산은 하지 않으며 판수는 직접 입력 (자세한 제약은 [docs/calculator-constraints.md](docs/calculator-constraints.md) 참고)
- **패로 계산하기** 탭: 손패·후로·오름패·장풍·자풍·화료유형·리치·도라/뒷도라·연장을 직접 입력하면 채점 엔진이 역·부수·점수를 자동 계산
    - 후로(치/퐁/가깡/밍깡/안깡)와 적도라 입력 지원
    - 자/친은 자풍으로 판정 (동=친)

## 점수표

판수×부수에 따른 점수 조견표입니다.

- 자/친 전환, 한 칸에 론(윗줄)·쯔모(아랫줄) 점수 동시 표기
- 만관 이상은 점수 대신 명칭(만관·하네만·배만·삼배만·헤아림 역만)으로 묶어 표시
- 기본은 20~60부만 노출, 70부 이상은 버튼으로 확장
- 성립하지 않는 칸(20·25부 1판 등)은 `—`로 표시

## 역 목록

작혼 룰 기준 역 정보를 판수 오름차순으로 정리한 목록입니다.

- 역별 판수·멘젠 한정·후로 시 쿠이사가리 태그 표시
- 역만/더블역만 그룹 분리
- 설명과 예시 패 렌더링
- 채점 엔진(`score.ts`)이 인정하는 역과 일치 (도라·적도라는 역이 아니므로 제외)

## 적용 룰 (작혼 기준)

- 연풍 또이쯔 4부
- 적도라 3장 (만·통·삭 5 각 1장)
- 더블역만: 국사무쌍 13면 / 스안커 단기 / 대사희 / 순정구련보등, 역만 복합 합산, 헤아림역만
- 고점법 채택
- **의도적 차이**: 연습하기 출제 시 뒷도라 미적용 (입력한 정보만으로 답이 정해지도록 —
  패로 계산하기에서는 뒷도라·연장 입력 가능), 키리아게 만관 미적용 (4판 30부 = 7700 그대로),
  상황역(하이테이·천화 등)은 엔진만 구현하고 출제 제외

자세한 설계는 [docs/score-trainer-plan.md](docs/score-trainer-plan.md) 참고.

## 패 이미지

패 이미지는 [FluffyStuff/riichi-mahjong-tiles](https://github.com/FluffyStuff/riichi-mahjong-tiles)의
SVG(퍼블릭 도메인, CC0)를 사용합니다. (`src/assets/tiles/`)

## 개발

```bash
npm install
npm run dev      # 개발 서버
npm test         # 채점 엔진 골든 케이스 + 생성기 합법성 + 점수표/스모크 테스트
npm run build    # 타입 검사 + 프로덕션 빌드
```

## 구조

```
src/core/       순수 로직 (UI 무관)
  tiles.ts        패 id·정렬·도라 매핑
  types.ts        공용 타입 정의
  decompose.ts    화료형 분해 (일반형/치또이/국사)
  score.ts        역 판정 + 부수 계산 (고점법)
  points.ts       점수 테이블·지불 분배
  generator.ts    문제 생성기
  yakuData.ts     역 목록 화면용 정적 역 데이터
src/features/   화면
  Home.tsx        홈 메뉴 (연습·계산 + 더 알아보기)
  ScoreTrainer.tsx  연습하기 (점수 계산 연습)
  Calculator.tsx    계산하기 (점수·부수 계산기)
  ScoreTable.tsx    점수표
  YakuList.tsx      역 목록
  FuGuide.tsx       부수 계산법
src/components/ 패 렌더링
src/App.tsx     라우팅
```
